## [Unreleased]

## [0.2.0]
### Added
- Serde `derive(Serialize,Deserialize)` for all types when using the `serde` feature.

## [0.1.1]
### Added
- `PartialEq<{float}>` for all types. Meaning one can write:
  ```rust
  let n = FiniteF32::new(1.0).unwrap();
  assert_eq!(n, 1.0);
  // instead of
  assert_eq!(n.get(), 1.0);
  ```
- Reexport `float_cmp::Ulps`

- Initial version
